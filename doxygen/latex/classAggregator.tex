\subsection{Detailed Description}
Class for aggregating functions. 

The \hyperlink{classAggregator}{Aggregator} incapsulates different aggregating functions to use on the result data of the benchmarks. The preferred function for your benchmark can be set directly in your benchmark class via the set\-Aggregating\-Function().
